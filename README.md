# IAMILKAY® Corporate Website

Production-ready Next.js corporate website for IAMILKAY®.

## Stack

- Next.js App Router
- Tailwind CSS
- Framer Motion
- Nodemailer contact API
- Vercel-ready project structure

## Local Development

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and configure SMTP variables.

All contact submissions are sent to `MAIL_TO`, defaulting to `info@iamilkay.co.uk`.
